AlexGames

Licenses

Back to AlexGames main
  • Lua (license) (web site): A wonderful scripting language whose interpreter compiles into about 400 kB of WASM.
  • Emscripten (license) (web site): Compiles C/C++ code into webassembly (WASM). This is invaluable to this project both for compiling my own C/C++ code, but also the Lua interpreter. It also packages my Lua files in a convenient filesystem-like way, and makes it easy for me to upload zip bundles of Lua files to the browser.
  • libzip (license) (web site): Handles unzipping user uploaded bundles of Lua scripts, to allow users to play their own games without having to build this project.
  • zlib (license) (web site): Dependency of libzip.
  • Python websockets library: (license) (web site): Used for websocket multiplayer server.
  • Android application:
    • nanohttpd: (license) (web site): Lightweight HTTP and websocket server, used for hosting an AlexGames server on your phone.
